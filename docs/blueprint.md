# **App Name**: FormFlow PDF Filler

## Core Features:

- PDF Template Upload: Allow users to upload a PDF template to use as a base form.
- Dynamic Input Field Creation: Enable users to add input fields dynamically onto the PDF template with precise positioning.
- Data Input: Provide input fields for users to enter data.
- PDF Export: Export the filled form as a PDF document.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) for a clean, professional look. This color evokes a sense of trust and stability, essential for a form-filling application. Its brightness works well against a white background.
- Background color: White (#FFFFFF) to ensure maximum readability and a minimalist aesthetic.
- Accent color: Steel blue (#4682B4) to highlight interactive elements and call-to-action buttons. This darker shade provides good contrast and draws attention where needed.
- Font: 'Inter', sans-serif, for all text elements. This provides a modern, neutral, and easily readable typeface, perfect for forms.
- Use a clean and intuitive layout that allows users to easily upload templates, add fields, and input data.
